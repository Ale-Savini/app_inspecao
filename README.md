# App Inspeções CEMIG

Aplicativo para controle de inspeções em subestações de energia.

## Funcionalidades

- Cadastro de subestações
- Cadastro de equipamentos
- Registro de inspeções
- Histórico de inspeções

## Tecnologias

- Flutter
- SQLite
- Material Design

## Como executar

```bash
flutter pub get
flutter run
